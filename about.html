About This Project

The Berkeley Budget Discipline Framework is a proposal for improving how budget decisions are made in Berkeley — not a campaign for or against any specific program.

What This Project Is

The Berkeley Budget Discipline Framework is a proposed decision-making framework for municipal budgeting. It asks that every budget proposal — regardless of source or constituency — be evaluated using the same six questions.

The framework is the flagship project of Berkeley Budget Lab, a civic initiative focused on improving transparency, evidence, and discipline in Berkeley's budget process. The Budget Lab also develops interactive tools and curates resources to help residents understand the city's finances.

What This Project Is Not

This project is not an advocacy campaign against specific programs or organizations. It does not promote predetermined budget outcomes. It does not tell Berkeley what to fund or cut.

The purpose of the framework is to ensure every proposal is evaluated using the same transparent standards. Reasonable people can disagree about individual budget decisions while agreeing that every public dollar deserves the same careful review.

Mission

A budget is not a list of good things.
It is a transparent ranking of priorities.

The objective is to build agreement around a budgeting framework before discussing specific budget decisions. Standards before examples. Process before personalities.

Guiding Principles

  • Standards before examples.
  • Process before personalities.
  • Evidence before conclusions.
  • Transparency over rhetoric.
  • One standard for every public dollar.
  • Good programs should welcome good scrutiny.
  • Reasonable people can disagree about outcomes while agreeing on process.
